Installation

Follow the steps below to get started with your Site Template:

  1. Open the Package/HTML Folder to find all the Templates Files
  2. You will need to Upload these files to your Web Server using FTP in order to use it on your Website.
  3. Make sure you upload the required files/folders listed below:
    • HTML/css - Stylesheets Folder
    • HTML/fonts - Fonts Folder
    • HTML/img - Images Folder
    • HTML/js - Javacripts Folder
    • HTML/php - php File
    • HTML/index.html - Index File/Homepage
    The other files can be used according to your preferences.
  4. You're now good to go..! Start adding your Content and show off your Brand New Beautiful Website in style.

Logo Settings

The Logo Container can be found in the Nav Container - .nav-brand

<a class="navbar-brand">
	<src="img/logo.png" alt="FOLIO LOGO">
</a>

Note The Logo Image's maximum height can be 100px. Also make sure you also add the Dark Logo which provides even more intuitiveness when you switch between Light/Dark Scheme. However, it is optional.

Theme Customization

We have include a Custom CSS File in the css Folder so that you can better handle your Customizations while updating folio to the Latest Version. Simply add all your Custom CSS Codes in the css/style.css File and link it in the Document <head> after the Linking. Also make sure that this is the Last Linked CSS File in the Document <head> so that your Custom CSS Styles are Overwritten properly.

Example:
<head>

	...

	<!-- Here goes your custom.css
	============================================= -->
	<link rel="stylesheet" href="css/style.css" type="text/css" />

</head>
